## Graceful Degradation and service workers

With service workers, we can ensure that our application addresses an even wider audience - as it will now address the lower level of a lack of a reliable network connection with grace. Now, loading a web-page once is all you need - post that, service workers ensure that you can have access to the site even without connection because of caching (through network interception).
